# Washington State Electric Vehicle Population Analysis

## Description
Tableau dashboard analyzing Washington state's EV registration data (~130K vehicles up to 2023). Explores adoption trends, geographic distribution, top makes/models, vehicle types, ranges, and clean fuel eligibility to highlight EV growth and policy insights.

## Tools Used
- Tableau Desktop for data modeling, calculations, and interactive visualizations.
- CSV dataset from Washington State Department of Licensing (public source).

## Key Insights
- **Growth**: EVs surged in 2021-2023 (~40% of registrations), with Tesla leading at 46%.
- **Geography**: Over 50% in King County (Seattle area dominant); maps show urban concentration.
- **Makes/Models**: Tesla (Model 3/Y/S top), Nissan Leaf, Chevy Bolt/Volt; BEVs ~77%.
- Interactive filters for make (e.g., KIA), year, type, and drill-downs by county/city.
